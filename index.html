关于我

👋 你好！ 我是 Clarity Wu， 一名嵌入式软件开发工程师，专注于 智能硬件 与 消费电子 领域。

熟悉从 裸机编程 到 实时操作系统（FreeRTOS / RT-Thread） 的完整开发流程，对 STM32、ESP32 等主流 MCU 有丰富的项目经验， 擅长驱动开发、传感器数据采集、通信协议（I2C / SPI / UART / CAN）以及电机控制。

相信好的嵌入式软件 = 对硬件的深入理解 + 严谨的工程实践。 热爱技术，享受从零到一将产品落地的过程。

STM32 FreeRTOS 传感器

智能手套与视觉定位联动系统

基于 STM32 的多传感器数据手套，集成柔性应变传感器与 IMU， 结合视觉定位实现手势识别与二自由度云台联动控制。 无线通信，上位机实时交互。

STM32 PID 电机控制

二自由度云台控制系统

基于 STM32 的舵机云台，采用 PID 闭环控制算法实现精确角度控制。 支持串口指令、预设轨迹跟踪，适用于视觉定位与目标追踪系统。

GitHub →
FreeRTOS ADC 信号处理

多路应变片数据采集系统

多通道应变片信号采集方案，使用 ADC 多路采样与数字滤波算法， 实现高精度力传感器数据采集，用于手势识别数据手套的触觉反馈。
